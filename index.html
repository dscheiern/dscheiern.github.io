Movie Madness

The intent of this program is to find a path between any two targets -- a movie or actor -- in the fewest steps possible. I constructed a Graph ADT using actor/movie data in a text file, where the nodes are the movies/actors and the edges are the connections between the nodes. The user inputs the text file name, then the actor or movie names when prompted. I used a breadth first search algorithm to find the shortest path between the two targets. The program then prints the path, if found, between the two targets. See the complete code here and here.

  • Language: Java
  • Date: April 2018
  • Topic: Graphs
