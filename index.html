Movie Madness
Language: Java
Topic: Graphs
The intent of this program is to find a path between any two targets -- a movie or actor -- in the fewest steps possible. I constructed a Graph ADT using actor/movie data in a text file, where the nodes are the movies/actors and the edges are the connections between the nodes. The user inputs the text file name, then the actor or movie names when prompted. I used a breadth first search algorithm to find the shortest path between the two targets. The program then prints the path, if found, between the two targets.

Mathematical Modeling Final Project
Optimizing MLB Pitching Using Markov Chains
Topic: Markov Chains, Transition Matrices, Optimization
View  Transition Matrix  and   final write-up
For the first part of my final project, I tried to solve my favorite MLB team's pitching problems. I used a Markov Chain with two absorbing states, and tried to optimize the probability of being absorbed at "Win." For the transition probabilities, I used the concept of predator-prey models to determine which pitching statistics should "help" or "harm" the probability of transitioning to that state. For the second part of my final project, I proved the theorem that the k-step transition matrix of a Markov Chain is the original transition matrix raised to the kth power.
